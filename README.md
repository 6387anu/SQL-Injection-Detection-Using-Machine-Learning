🔐 SQL Injection Detection Using Machine Learning

A Machine Learning–powered Flask web application that detects SQL Injection (SQLi) attacks in user input.
This system uses intelligent ML classification instead of traditional rule-based filtering to provide smarter and more reliable security for web applications.

📌 Project Description

SQL Injection is one of the most dangerous web vulnerabilities. Attackers manipulate SQL queries to:

Bypass authentication

Access sensitive data

Modify or delete records

Compromise backend systems

This project uses Machine Learning + Flask to automatically classify inputs as:

✅ Safe Input

⚠️ SQL Injection Attack

🎯 Features

Flask-based web interface

ML-based SQL injection detection

Real-time prediction

Intelligent classification

NLP-based feature extraction

High accuracy detection

Easy integration with web apps
